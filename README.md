# RestaurantOrderingSystem
Automatic Restaurant Ordering Management System built with Core Java and MySQL using JDBC — features order placement, kitchen queue automation, and invoice generation.


## 🍽️ Automatic Restaurant Ordering Management System

A console-based backend application built with **Core Java** and **MySQL** that automates 
the complete restaurant workflow — from customer order placement to kitchen updates 
and automated bill generation.

### 🔧 Tech Stack
- Java (Core Java, OOP)
- MySQL (JDBC connectivity)
- IDE: IntelliJ IDEA / Eclipse

### ✨ Features
- 📋 Dynamic menu management (Add, Update, Disable items)
- 🛒 Customer order placement with real-time validation
- 👨‍🍳 Automatic kitchen queue update on every order
- 🧾 Auto invoice generation with tax calculation
- 🔒 Transaction-safe order saving (commit/rollback)
- 📦 Modular DAO + Service architecture
